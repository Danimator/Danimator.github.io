Inferno Chess Engine

C++11 | August 2016 - Present | Executable

A C++ Chess AI written around the Principal Variation search algorithm, which utilizes multithreading, magic bitboards, a transposition table, and multiple heuristics to improve playing strength. It supports the UCI protocol, and plays with a strength of approximately 1900-2000 Elo CCRL in Blitz.

Chess Clock

HTML CSS JavaScript | March 2017 | Github Demo

A chess clock which operates on a mobile-friendly website. Optimized to support both landscape and portrait phone orientations, it also detects multi-touch to emulate a real chess clock. Features customizable time control for each player, as well as three (Fischer, Bronstein, and Simple) time compensation types.
